# Demo Script

1. Run `npm run dev`.
2. Open `http://localhost:3456`.
3. Pair the session using code `482913`.
4. Press `Start` and point out the live transcript and increasing WAV chunk count.
5. Press `Mark Insight`.
6. Press `Simulate network drop`.
7. Show the amber `OFFLINE — AUDIO SAFE` banner and growing queue depth.
8. Press `Reconnect network`.
9. Show `SYNCING`, then queue depth back to zero.
10. Press `Stop` and open the local report.
11. Close with: `The venue internet failed. The panel didn't.`
